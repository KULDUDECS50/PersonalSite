My Recent Projects

PoliticianWatch (2023)

My submission for the MEGA Hackathon 2023, Anti-Corruption Category

Our goal is to allow the general public to have easily accessible information on corruption and promote political transparency globally. PoliticianWatch uses an algorithm to sort through data from multiple sources and databases and cross references and checks them to create a comprehensive view on those corrupted For those politicians who have a spotlight, some detail will be provided into them as well. I was able to get this data by UN APIs with Python and stoed it in Supabase + Firebase. Front End was developed using HTML5, CSS, Javascript.

Publications

API Terminology Handbook (2023)

The API Terminology Handbook is a comprehensive 42-page guide to understanding the terminology associated with APIs, or application programming interfaces. The handbook covers topics such as the different types of APIs, how they work, and best practices for working with them. It also includes tips on how to use the terminology to set yourself apart from other developers, as well as how to support the author's work. The handbook is a valuable resource for developers who want to stay up-to-date with the latest technologies and gain a deeper understanding of APIs.
